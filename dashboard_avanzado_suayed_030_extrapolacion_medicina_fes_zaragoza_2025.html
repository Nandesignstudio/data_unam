Resumen ejecutivo

El hallazgo SUAyED se incorpora como una tasa externa de cola alta: la proporción esperada de sustentantes con 115 a 120 aciertos. Para este dataset, usando como denominador a quienes presentaron examen, el valor esperado es:

H115–120 = N × p = 3,943 × 0.003 = 11.83

El dataset original tiene 13 aspirantes en 115–120, equivalente a 0.33%. Es decir, el dato observado está muy cerca del hallazgo SUAyED de 0.30%.

Lectura central: el hallazgo SUAyED no empuja por sí mismo el corte hacia arriba; más bien funciona como control de plausibilidad. El dataset original ya es compatible con una cola alta de alrededor de 0.30%.

Metodología estadística

  1. Se define el evento de cola alta: obtener 115–120 aciertos.
  2. Se usa el hallazgo empírico SUAyED como parámetro: p = 0.003.
  3. Se modela el número de sustentantes de cola alta como Binomial(N, p).
  4. Se compara el número observado contra el esperado, usando desviación estándar y z-score.
  5. Se evalúan escenarios de estrés: duplicar, triplicar y reformular la cola alta.

Conclusión técnica

La integración del hallazgo SUAyED permite pasar de un ejercicio arbitrario de multiplicación a un análisis con control externo de plausibilidad. Bajo el parámetro 0.30%, esperaríamos alrededor de 11.8 sustentantes con 115–120 aciertos; el dataset original tiene 13, por lo que es compatible con ese hallazgo.

Los escenarios de duplicación y triplicación siguen siendo útiles como pruebas de estrés, pero no deberían presentarse como predicción central. La predicción central restringida por SUAyED conserva un corte cercano al observado, mientras que el escenario reformulado + triplicado lleva el corte a 116 aciertos, pero con una cola alta de 87 sustentantes, mucho mayor al 0.30% esperado.

Recomendación metodológica: reportar dos capas. Primero, el modelo central calibrado por SUAyED 0.30%; segundo, escenarios de estrés —duplicado/triplicado— claramente etiquetados como contrafactuales extremos.
